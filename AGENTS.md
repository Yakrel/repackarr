## Quality Assurance (CI/CD)
The parsing logic is critical. To ensure stability:
- **Mandatory Test:** Before every commit or push, run:
  ```bash
  npx tsx tests/test_parsing.ts
  ```
- **Reference Data:** The golden dataset is stored in `tests/torrent_examples.json`.
- **Target:** Always maintain **100% Accuracy** on this dataset.

## Database Migrations
This project uses Drizzle Kit for database migrations. 
- **When modifying schema:**
  ```bash
  pnpm db:generate
  git add drizzle/ src/lib/server/schema.ts
  ```
- Never modify SQL files in `drizzle/` manually.
- Automatic migration runs on app startup.
